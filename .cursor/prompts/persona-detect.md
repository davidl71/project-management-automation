# Persona Detection

Help me determine the right workflow for the user.

## Persona Keywords
- **Developer**: code, implement, fix, build, debug
- **Project Manager**: sprint, planning, status, delivery, blockers
- **Code Reviewer**: PR, review, approve, merge, quality
- **Architect**: design, architecture, coupling, patterns
- **Security**: vulnerability, security, CVE, audit
- **QA**: test, coverage, quality, defect
- **Executive**: status, summary, overview, stakeholder
- **Tech Writer**: documentation, docs, readme

## Actions by Persona
- Developer → `/exarp/prompt persona_developer`
- PM → `/exarp/prompt persona_project_manager`
- Reviewer → `/exarp/prompt persona_code_reviewer`
- Architect → `/exarp/prompt persona_architect`
- Security → `/exarp/prompt persona_security`
- QA → `/exarp/prompt persona_qa`
- Executive → `/exarp/prompt persona_executive`
- Tech Writer → `/exarp/prompt persona_tech_writer`
